A **widget** is the core building block in Acme. Each widget is a small, embeddable unit
you can configure and drop onto any web page.

## Create a widget

1. From the dashboard, click **New widget**.
2. Give it a name and pick a template to start from.
3. Configure its options in the right-hand panel — changes preview live.
4. Click **Save**.

## Publish

1. Open the widget and click **Publish**.
2. Copy the embed snippet shown in the dialog.
3. Paste it into your site's HTML where you want the widget to appear.

Published widgets update automatically — edit and re-save, and live embeds pick up the
change within a minute.

## Tips

- **Use templates** — start from a template and customize, rather than building from scratch.
- **Preview before publishing** — the live preview reflects exactly what visitors will see.
- **Version safely** — publishing never breaks an existing embed; the snippet stays the same.

## Where to find it

In the app: **Widgets → New widget**.
